# Prisma + Express.js + PostgreSQL

This guide covers the steps to set up and configure Prisma ORM with an Express.js application using a PostgreSQL database.

## Prerequisites
- Node.js installed
- PostgreSQL database running and accessible

## 1. Project Initialization
Initialize your Node.js project and install the required dependencies:

```bash
# Initialize a new Node.js project (if not already done)
npm init -y

# Install Express.js
npm install express

# Install Prisma CLI as a development dependency
npm install prisma --save-dev

# Install Prisma Client
npm install @prisma/client
```

## 2. Initialize Prisma
Run the following command to create the initial Prisma setup. This will create a `prisma` folder with a `schema.prisma` file and an `.env` file for your database connection:

```bash
npx prisma init
```

## 3. Configure Database Connection
Open the `.env` file generated by Prisma and update the `DATABASE_URL` with your PostgreSQL connection string:

```env
# Format: postgresql://USER:PASSWORD@HOST:PORT/DATABASE?schema=public
DATABASE_URL="postgresql://myuser:mypassword@localhost:5432/mydatabase?schema=public"
```

## 4. Define Your Schema
Open `prisma/schema.prisma` and define your database models. For example, a simple `User` model:

```prisma
generator client {
  provider = "prisma-client-js"
}

datasource db {
  provider = "postgresql"
  url      = env("DATABASE_URL")
}

model User {
  id        Int      @id @default(autoincrement())
  email     String   @unique
  name      String?
  createdAt DateTime @default(now())
}
```

## 5. Run Migrations
Whenever you change your schema, run a migration to update the actual PostgreSQL database schema:

```bash
npx prisma migrate dev --name init
```

## 6. Generate Prisma Client
Generate the Prisma Client to interact with your database using TypeScript/JavaScript:

```bash
npx prisma generate
```

## 7. Use Prisma in Express
Here is a basic example of how to use the Prisma Client inside an Express application (e.g., in `src/app.js` or `server.js`):

```javascript
const express = require('express');
const { PrismaClient } = require('@prisma/client');

const prisma = new PrismaClient();
const app = express();

app.use(express.json());

// Create a new user
app.post('/users', async (req, res) => {
  try {
    const { name, email } = req.body;
    const user = await prisma.user.create({
      data: { name, email },
    });
    res.status(201).json(user);
  } catch (error) {
    res.status(400).json({ error: 'Failed to create user' });
  }
});

// Get all users
app.get('/users', async (req, res) => {
  const users = await prisma.user.findMany();
  res.json(users);
});

const PORT = process.env.PORT || 3000;
app.listen(PORT, () => {
  console.log(`Server is running on http://localhost:${PORT}`);
});
```
